# ねこよみ堂（NEKOYOMIDO）— 猫の占いポータル

「猫×占い」をテーマにした無料の占い・診断ポータルサイト。完全静的（サーバー・DB不要）。

## サイト構成

```
index.html            … TOPページ「ねこよみ堂」（占いメニュー一覧）
necogram/index.html   … ネコグラム（16猫種×4スタイル=64タイプ性格診断＋相性診断）
omikuji/index.html    … ねこおみくじ（メインコンテンツ。気分5問→1日1回のおみくじ＋気分別アドバイス・8ランク・猫大吉3%・連続おまいり記録つき）
ranking/index.html    … ねこ運勢ランキング（星座×血液型=48匹。日付シードで毎日0時に総入れかえ。全員が同じ表を見る公平なくじ。自分の星座・血液型は端末に記憶）
cards/index.html      … ねこの森カード（ルノルマン風・オリジナル36枚。1日1枚、フリップ演出、カード一覧はSEOコンテンツ兼用）
numbers/index.html    … ねこ数字占い（携帯下4桁の合計0〜36の「猫数字」辞典37種。何度でも占える回遊コンテンツ）
```

- 今後の占いは同じようにサブフォルダを追加して、TOPのメニューカードの `soon` クラスを外してリンクするだけ
- 近日公開枠（実装待ち）：365日たんじょう日ねこ図鑑

## デザインシステム（ケルティック・ミスティック）

共通テーマ「森のおくの占いのやかた」。全ページで統一：
- 配色：深い森の緑（#0e1a15）×古金（#c9a961）×月光クリーム（#e9dfc6）
- 星屑の夜空背景（CSS radial-gradient）、二重の金枠パネル、トリケトラ（三つ組結び目）飾り
- 欧文は Uncial Antiqua（ケルト写本風）、和文見出しは Shippori Mincho
- CSS変数で管理。新ページは既存ページの `:root` と `.card` `.rule` `.btn` をコピーすれば統一される

## サイト名「ねこよみ堂」について（市場調査済み）

「ねこ占い」「猫占い」「にゃんこ占い」は既存サービス多数（nyankouranai.com、さちこい、占いレディ等）のため回避。
「ねこよみ堂」は2026年7月時点の検索で同名の占いサービスは確認されず。独自ドメインは nekoyomido.com 等を取得時に空き確認を。

## 画像アセット（Adobe Firefly生成・組み込み済み）

- `assets/img/` … 本番用WebP 95枚（森カード36・猫種16・運勢8・スタイル4・星座12・血液型4・数字7・OGP8）＋ `thumb/` に軽量サムネ。全ページ組み込み済み
- `assets/prompts.md` … 生成プロンプト表（リテイク・追加生成時に使用）
- `assets/review/` … 選定作業用（picks.json等）。**デプロイ時にアップロード不要**
- OGP画像のmetaタグは相対パスで設定済み。**デプロイ後にドメイン付き絶対URLへ書き換えること**（各HTMLの`og:image`のコメント参照）
- 軽微な既知課題（気になったらリテイク）：スコティッシュフォールドの折れ耳が出ていない／おみくじOGPに祠がない／card06・07・09・11は猫不在の風景 など`assets/review/view.html`の注記参照

## Cloudflareへのデプロイ（無料）

1. https://dash.cloudflare.com → Workers & Pages → Create → Pages
2. 「Upload assets」でこのフォルダごとドラッグ＆ドロップ（フォルダ構成そのまま）
3. `*.pages.dev` で即公開。AdSense審査のため独自ドメイン推奨

## 広告収入のおすすめ（優先順）

1. **Google AdSense（本命）** — クリック単価が国内他社の約2倍以上。`AD SPACE` の div に広告コードを貼るだけ（TOP2枠＋ネコグラム4枠）
2. **忍者AdMax** — 審査なし。AdSense審査待ちのつなぎに
3. **アフィリエイト併用** — A8.net／もしもアフィリエイトで猫グッズ・キャットフードを「タイプ別おすすめ」として設置。猫好き特化サイトとの相性抜群
4. 収益拡大後は Google Ad Manager 等へ

### 収益を最大化するコツ
- 一番読まれるのは診断結果の直下（配置済み）
- 占いを増やすほど回遊→PV→広告収益が伸びる構造。TOPページがそのハブ

## 毎日来たくなる仕組み（実装済み）
- TOPヒーローの最優先CTA＝「きょうのねこおみくじをひく」。ひいた後は「きょうの結果をみる『大吉』」に自動で変わる（localStorage参照）
- 連続おまいり記録（streak）：毎日ひくと連続日数が伸びる。1日あくとリセット。3日以上でシェア文にも連続日数が入る
- 気分の5問：ひく前に「目ざめ・こころの天気・エネルギー・気になること・どんな一日にしたいか」を質問。運勢とは別に「きょうのこころ模様」ラベル＋関心事×元気度のアドバイス＋過ごし方ヒントを表示
- アドバイスは各スロット8パターン（4テーマ×3段階×8=96本、過ごし方4×8=32本、計128本）。年初からの通し日数でローテーションするため、連日で同じ文には絶対にあたらず、8日間で全パターンを一巡する
